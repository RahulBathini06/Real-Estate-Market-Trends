# 🏠 Real Estate Market Analysis Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard designed to analyze real estate market trends and property listings. It provides valuable insights into property prices, locations, property types, and other key metrics to help users understand market performance and make informed decisions.

## 📊 Dashboard Features
- 📍 Property Price Analysis
- 🏘️ Top Neighborhoods by Average Price
- 🏢 Property Type Distribution
- 📏 Average Property Size Analysis
- 💰 Price Comparison by Location
- 📈 Interactive Filters and Slicers
- 📊 KPI Cards for Key Metrics
- 🌍 Location-wise Property Insights

## 🛠️ Tools Used
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Microsoft Excel / CSV Dataset


## 📈 Key Insights
- Identified the most expensive neighborhoods.
- Compared average property prices across different locations.
- Analyzed property size and pricing trends.
- Examined the distribution of different property types.
- Created interactive visualizations for easy data exploration.
